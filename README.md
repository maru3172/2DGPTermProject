# 2DGPTermProject
2D Game Programming Term Project 2021180023

Project 2021180023

게임의 간단한 소개

"1942"와 유사한 생존 게임으로, 플레이어가 전투기를 조종하여 몰려오는 적의 공격을 피하는 게임,

주요 기능은 플레이어 조작과 충돌 판정, 가까운 적에게 유도되는 탄이다.

목표 변경 사유: 게임이 너무 단조롭고 차별화된 점이 없다 판단하여 수정
